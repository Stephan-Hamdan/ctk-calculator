# CustomTkinter Calculator

A minimal Python GUI calculator built with **CustomTkinter**.  
Supports one-operator expressions like `12+5`, `9-3`, `4*2`, `8/4`.

## Features
- Digit and operator buttons (`+ - * /`)
- Decimal point
- `=` to evaluate
- `CL` to clear
- Compact, beginner-friendly code
